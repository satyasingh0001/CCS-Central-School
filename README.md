# CCS-Central-School
CCS Central School is a full-stack School Management System (ERP) designed to manage administrative, teacher, and student operations in a modern and secure way. The project is built using the MERN stack (MongoDB, Express, React, Node.js) and follows industry-level architecture with role-based access control.
Features
-> Public Website

Home, About, Academics, Admission, Faculty, Gallery, Contact

Responsive design for all devices

Clean UI with modern navigation

 -> Authentication & Security

JWT-based authentication

Role-based access control (Admin / Teacher)

Protected routes for admin panel

Secure login & logout flow

-> Admin Panel

Admin dashboard with attendance analytics

Teacher attendance management

Student attendance management

Monthly attendance calendars

Student attendance dashboard

Breadcrumb navigation & active sidebar highlighting

-> Teacher Panel

Teacher dashboard with monthly stats

Mark and view attendance

Restricted access based on role

-> Student Management

Class-wise student attendance

Daily attendance marking

Monthly student attendance calendar

Present  / Absent  indicators
